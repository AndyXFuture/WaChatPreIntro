# WaChatPreIntro
## 关于该项目

学习Android应用开发的一个期末项目：仿微信布局的个人介绍

使用`ViewPager`做的主页，可以滑动

简历页数据从资料页填入数据库生成

已签名打包APK，可直接安装手机上
